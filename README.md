# FMOE

![fmotool](./img/fmotool.gif)

## installation

1. clone repository to ~/moefiles

    ```
    git clone https://github.com/philopon/FMOE.git ~/moefiles/FMOE
    ```

2. load loader.svl from \$HOME/moefiles/start.svl

    for example:

    ```
    local function main []
        run '$HOME/moefiles/FMOE/loader.svl';
    endfunction
    ```
