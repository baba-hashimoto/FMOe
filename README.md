==
![fmotool](./img/fmotool.gif)

installation
--
1. clone repository to ~/moefiles

    ```
    git clone https://github.com/philopon/FMOE.git ~/moefiles/FMOE
    ```

2. load main.svl from $HOME/moefiles/start.svl

    for example:

    ```
    local function main []
        load '$HOME/moefiles/FMOE/main.svl';
    endfunction
    ```
